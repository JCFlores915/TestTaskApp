## Clonar el repositorio

```bash
git clone https://github.com/JCFlores915/TestDevsu.git](https://github.com/JCFlores915/TestTaskApp.git
```

## Instalar dependencias

```bash
npm install
```

## Correr el proyecto en Android

```bash
npm run android
```

## Correr los test

```bash
npm test
```

## Tecnologias usadas

- React Native 
- React Navigation
- Typescript
- Jest
